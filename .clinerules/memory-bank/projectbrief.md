# 项目基础文档 (Project Brief)

## 项目概述

**项目名称**: 张晋哲 - 现代化个人简历网站  
**项目类型**: 个人在线简历网站  
**项目目标**: 创建一个现代化、响应式、高性能的个人简历展示网站

## 核心需求

- **个人信息展示**: 完整展示个人基本信息、工作经历、项目经验
- **技能展示**: 以可视化方式展示技术技能和能力水平
- **响应式设计**: 支持桌面端、平板端、移动端的完美适配
- **性能优化**: 快速加载、良好的 SEO 表现、用户体验流畅
- **现代化 UI**: 采用现代设计原则，简洁优雅的视觉效果

## 功能需求

1. **个人信息模块**: 基本信息、联系方式、个人简介
2. **技能展示模块**: 技术栈、熟练程度可视化展示
3. **工作经历模块**: 时间线式工作经历展示
4. **项目经验模块**: 项目卡片式展示，包含技术栈和描述
5. **教育背景模块**: 学历信息展示
6. **社交媒体模块**: 社交链接和联系方式
7. **导航系统**: 平滑滚动导航、锚点定位

## 技术要求

- **前端框架**: Next.js 15 (App Router)
- **编程语言**: TypeScript
- **样式框架**: Tailwind CSS
- **动画库**: Framer Motion
- **UI 组件**: Radix UI
- **构建工具**: Turbopack

## 目标用户

- 潜在雇主和招聘人员
- 技术同行和合作伙伴
- 个人品牌展示的访客

## 成功指标

- 页面加载速度快 (FCP < 1.2s)
- 良好的 SEO 表现
- 响应式适配完美
- 用户体验流畅
- 信息展示清晰完整
